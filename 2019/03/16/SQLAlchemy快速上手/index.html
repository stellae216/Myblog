SQLAlchemy简介

SQLAlchemy是Python编程语言下的一款ORM框架，该框架建立在数据库API之上，使用关系对象映射进行数据库操作，简言之便是：将对象转换成SQL，然后使用数据API执行SQL并获取执行结果。

SQLAlchemy本身无法操作数据库，其必须以来pymsql等第三方插件，从而实现对数据库的操作，格式如下：

一. 绑定数据库

使用 Engine/ConnectionPooling/Dialect 进行数据库操作，Engine使用ConnectionPooling连接数据库，然后再通过Dialect执行SQL语句。

二. ORM映射

使用 ORM/Schema Type/SQL Expression Language/Engine/ConnectionPooling/Dialect 所有组件对数据进行操作。根据类创建对象，对象转换成SQL，执行SQL。

四. 软删除查询

使用orm查询时自动过滤软删除字段deleted，支持sqlalchemy或者flask_sqlalchemy方式查询

  • __new__：添加需要过滤的字段进行查询，对join和outerjoin都有效，但是不会对被join的表过滤。
  • get ：针对falsk_sqlalchemy.get方法实现过滤。使用了另外一种方式实现，查询时不过滤，根据查询结果，如果已经软删除，则返回None。
  • BaseQuery: 这里我们继承flask_sqlalchemy.BaseQuery，如果不需要使用flask_sqlalchemy实现的paginate等方法，继承对象替换为sqlalchemy.orm.Query即可。

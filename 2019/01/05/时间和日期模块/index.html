时间和日期模块

本文最后更新于：1 年前

time模块

time模块中的几种类型：

  • 时间戳： 1547729670.5185406 - 距离 1970-01-01 00：00：00 时间的总秒数
  • 时间元组：time.struct_time(tm_year=2016, tm_mon=7, tm_mday=21, tm_hour=22, tm_min=32, tm_sec=51，tm_wday=3, tm_yday=203, tm_isdst=0)
  • 英文显示：'Mon Jan 26 00:52:24 1970'
  • 格式化显示 ：'%Y-%m-%d %H:%M:%S'

先来看一下time模块的类型关系图：

常用命令

time.time() - 获取当前时间戳

time.localtime() - 获取当前时间元组

time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()) - 获取当前格式化显示

类型转换

  1. 时间戳 -> 时间元组
    time.localtime() - 获取当前时间元组

    时间元组可以进行运算，利用days() / seconds() / total_seconds()获取值，区别见下文

  2. 字符串 -> 时间元组
    time.strptime(str, '%Y-%m-%d %H:%M:%S')

  3. 时间元组 -> 英文显示
    time.asctime()

  4. 时间戳 -> 英文显示
    time.ctime()

  5. 时间元组 -> 格式化显示
    time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()) - strftime -> str-format-time

  6. 格式化显示 -> 时间元组

    time.strptime('2019-1-18 10:10:24', '%Y-%m-%d %H:%M:%S') - strptime -> str-parse-time

datetime模块

datetime中的类型时间元组形式: datetime.datetime(2019, 1, 17, 21, 37, 2, 766520)

常用命令

datetime.datetime.now() - 获取当前时间元组

datetime.datetime.now().timestamp() - 获取当前时间戳

datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S') - 获取当前格式化字符串

datetime.datetime.now().strftime('%x %X') - 获取当前格式化字符串

类型转换

  1. 时间元组 -> 英文显示(ctime)

    datetime.datetime.now().ctime()

  2. 时间元组 -> 格式化显示

    datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')

  3. 英文显示(ctime) -> 时间元祖

  4. 字符串 -> 时间元组

python-dateutil模块

日期解析器：能够将字符串转换为日期格式（datetime.datetime类型）

说明：

  1. days：来获取时间差的天数；
  2. seconds：来获取时间差中的秒数。注意，seconds获得的秒只是时间差中的小时、分钟和秒部分的和，并没有包含时间差的天数；
  3. total_seconds：来获取准确的时间差，并将时间差转换为秒。

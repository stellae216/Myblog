Swagger快速上手指南

本文最后更新于：2 年前

Swagger简介

Swagger是REST APIs文档生成工具。 主要的工作量应该在于Swagger文档的编写，而文档的编写需要遵循一定的规范，即Swagger API Spec。Swagger API Spec是Swagger用来描述REST API的语言，可以用YAML和JSON表示。

书写接口描述文档是一件十分痛苦的事情，但是当我们使用Swagger并按照一定规范描述项目接口，可以自动生成接口描述文档，解放生产力。查看Swagger描述文档有两种方式：

  1. 文档默认地址为项目运行根目录。
  2. 打开<https://petstore.swagger.io/> 网址，将项目生成json文件地址填入即可，如<https://xxx.io/api/swagger.json>

Flask-Restplus

flask-restplus是在flask框架下基于swagger的实现，我们可以使用flask-restplus代替flask-restful，其用法和后者类似，flask-restplus封装了很多方法可以对接口进行修饰。

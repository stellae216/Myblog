scrapy支持提取数据的几种方式：

  • CSS
  • XPATH
  • RE（正则）

Scrapy分布式

不同的主机的调度器从共享队列中获取爬取队列。最好使用redis列表或有序集合，提供重复过滤，调度器会从redis队列中取没有爬取的继续爬取。

redis本身不支持分布式，安装scrapy-redis

注意：爬取下来的数据不要存在本机，应该存放在共同的位置

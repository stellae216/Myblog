单元测试 - 针对程序中最小的功能模块(函数和方法)的测试

测试方法：

  • 白盒测试：程序员自己写的测试
  • 黑盒测试：测试人员或QA，不知道代码实现细节，只关注功能

执行单元测试

  • Python内置

    1. unittest.main()
    2. python3 -m unittest <文件名>

  • 第三方库

    自动检查执行单元测试

    • pytest

      安装 pip install pytest pytest-cov

      执行pytest 或者 查看测试详细信息pytest -v --cov

    • nose2

      安装 pip install nose2 cov-core

      执行nose2 或者 查看测试详细信息 nose2 -v -C

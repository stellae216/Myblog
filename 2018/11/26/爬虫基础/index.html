HTTP基本原理

URI和URL

URI (Uniform Resource Identifier) 统一资源标志符

URL(Universal Resource Locator) 统一资源定位符

HTTP和HTTPS

HTTP (Hyper Text Transfer Protocol) 超文本传输协议

HTTPS (Hyper Text Transfer Protocol over Secure Socket Layer) HTTP加入SSL层，传输内容通过SSL加密

  • 安全通道保证数据传输安全
  • 确认网站真实性

urllib库

urllib是基于http的高层库，它有以下四个主要功能：

  1. request处理客户端的请求
  2. response处理服务端的响应
  3. parse会解析url
  4. 主要用来识别网站的robots.txt文件，用得较少
